#######################
UC-U-007 活動を再開する
#######################

====
概要
====

休止中の活動を再開する。

========
アクター
========

ユーザー

========
前提条件
========

活動が休止中である。

========
事後条件
========

活動が進行中である。

========
トリガー
========

ユーザーが活動の再開をシステムに指示する。

==========
基本フロー
==========

#. ユーザーは再開する活動を指定する。
#. システムは活動を進行中にする。

==========
例外フロー
==========

なし。
