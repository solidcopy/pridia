###########################
UC-U-027 定期購入を登録する
###########################

====
概要
====

毎月、毎年に支払をする定期購入を登録する。

========
アクター
========

ユーザー

========
前提条件
========

登録する定期購入がある。

========
事後条件
========

定期購入が登録されている。

========
トリガー
========

ユーザーが定期購入の登録をシステムに指示する。

==========
基本フロー
==========

#. ユーザーは定期購入の内容を提示する。
#. システムは定期購入を登録する。

==========
例外フロー
==========

なし。

==============

====================
関連するユースケース
====================

なし。
