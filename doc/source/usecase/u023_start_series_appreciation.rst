#############################
UC-U-023 鑑賞期間を開始する
#############################

====
概要
====

アニメや連続ドラマなどのシリーズものの番組や小説の鑑賞を開始する。

一日で鑑賞が終わる映画などとは区別する。

========
アクター
========

ユーザー

========
前提条件
========

鑑賞を始めた作品がある。

その作品を過去に鑑賞したことはない。

========
事後条件
========

作品と鑑賞期間が記録されている。

========
トリガー
========

ユーザーが鑑賞開始の記録をシステムに指示する。

==========
基本フロー
==========

#. ユーザーは作品の内容を提示する。
#. ユーザーは初回の鑑賞について提示する。
#. システムは作品、鑑賞期間、初回の鑑賞を記録する。

==========
例外フロー
==========

なし。
