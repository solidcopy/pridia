###########################
UC-U-020 所有物を訂正する
###########################

====
概要
====

所有物についての情報を訂正する。

========
アクター
========

ユーザー

========
前提条件
========

訂正する所有物がある。

========
事後条件
========

所有物が訂正されている。

========
トリガー
========

ユーザーが所有物の訂正をシステムに指示する。

==========
基本フロー
==========

#. ユーザーは訂正する所有物を指定する。
#. ユーザーは所有物の訂正内容を提示する。
#. システムは所有物を訂正する。

==========
例外フロー
==========

なし。

==============
ビジネスルール
==============

なし。
