#########################
UC-U-011 進捗を訂正する
#########################

====
概要
====

記録した進捗に文章を追記したり誤りを訂正する。

========
アクター
========

ユーザー

========
前提条件
========

訂正する進捗が記録されている。

========
事後条件
========

進捗が訂正されている。

========
トリガー
========

ユーザーが進捗の訂正をシステムに指示する。

==========
基本フロー
==========

#. ユーザーは進捗を指定する。
#. ユーザーは進捗の訂正内容を提示する。
#. システムは進捗を訂正する。

==========
例外フロー
==========

なし。
