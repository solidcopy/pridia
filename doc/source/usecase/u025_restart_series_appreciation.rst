#################################
UC-U-025 鑑賞期間を再度開始する
#################################

====
概要
====

過去に鑑賞したことのある作品を再び鑑賞し始める。

========
アクター
========

ユーザー

========
前提条件
========

鑑賞を始めた作品がある。

その作品を過去に鑑賞したことがある。

========
事後条件
========

鑑賞期間が記録されている。

========
トリガー
========

ユーザーが鑑賞開始の記録をシステムに指示する。

==========
基本フロー
==========

#. ユーザーは鑑賞を開始した作品を指定する。
#. ユーザーは初回の鑑賞について提示する。
#. システムは鑑賞期間、初回の鑑賞を記録する。

==========
例外フロー
==========

なし。
