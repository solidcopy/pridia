#######################
UC-U-005 活動を開始する
#######################

====
概要
====

活動を開始する。

========
アクター
========

ユーザー

========
前提条件
========

開始する活動が登録されている。

活動が未着手である。

========
事後条件
========

活動が進行中である。

========
トリガー
========

ユーザーが活動の開始をシステムに指示する。

==========
基本フロー
==========

#. ユーザーは開始する活動を指定する。
#. システムは活動を進行中にする。

==========
例外フロー
==========

なし。
