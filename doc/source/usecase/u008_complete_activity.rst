#######################
UC-U-008 活動を完了する
#######################

====
概要
====

進行中の活動を完了する。

========
アクター
========

ユーザー

========
前提条件
========

活動が進行中である。

========
事後条件
========

活動が完了している。

========
トリガー
========

ユーザーが活動の完了をシステムに指示する。

==========
基本フロー
==========

#. ユーザーは完了する活動を指定する。
#. システムは活動を完了にする。

==========
例外フロー
==========

なし。
