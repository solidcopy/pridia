#############################
UC-U-024 鑑賞期間を完了する
#############################

====
概要
====

作品を最後まで鑑賞して鑑賞期間を終える。

========
アクター
========

ユーザー

========
前提条件
========

鑑賞を終えた作品がある。

========
事後条件
========

鑑賞期間の完了が記録されている。

========
トリガー
========

ユーザーが鑑賞完了の記録をシステムに指示する。

==========
基本フロー
==========

#. ユーザーは完了した鑑賞期間を指定する。
#. システムは鑑賞期間の完了を記録する。

==========
例外フロー
==========

作品を最後まで鑑賞するのを断念した
**********************************

#. ユーザーは鑑賞を中止したことを提示する。
#. システムは鑑賞期間の中止を記録する。
