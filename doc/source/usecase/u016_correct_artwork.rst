#########################
UC-U-016 作品を訂正する
#########################

====
概要
====

作品の記録の誤りを訂正する。

========
アクター
========

ユーザー

========
前提条件
========

訂正するべき作品がある。

========
事後条件
========

作品が訂正されている。

========
トリガー
========

ユーザーが作品の訂正をシステムに指示する。

==========
基本フロー
==========

#. ユーザーは訂正する作品を指定する。
#. ユーザーは作品の訂正内容を提示する。
#. システムは作品を訂正する。

==========
例外フロー
==========

なし。

==============
ビジネスルール
==============

なし。
