###########################
UC-U-028 定期購入を更新する
###########################

====
概要
====

定期購入の内容を更新する。

========
アクター
========

ユーザー

========
前提条件
========

更新する定期購入がある。

========
事後条件
========

定期購入が更新されている。

========
トリガー
========

ユーザーが定期購入の更新をシステムに指示する。

==========
基本フロー
==========

#. ユーザーは定期購入の更新内容を提示する。
#. システムは定期購入を更新する。

==========
例外フロー
==========

なし。
