#########################
UC-U-012 進捗を削除する
#########################

====
概要
====

進捗の記録自体が誤りであった場合に削除する。

========
アクター
========

ユーザー

========
前提条件
========

削除する進捗が記録されている。

========
事後条件
========

進捗が削除されている。

========
トリガー
========

ユーザーが進捗の削除をシステムに指示する。

==========
基本フロー
==========

#. ユーザーは進捗を指定する。
#. システムは進捗を削除する。

==========
例外フロー
==========

なし。
