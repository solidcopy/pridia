#########################
UC-U-015 鑑賞を訂正する
#########################

====
概要
====

鑑賞の記録の誤りを訂正する。

========
アクター
========

ユーザー

========
前提条件
========

訂正するべき鑑賞がある。

========
事後条件
========

鑑賞が訂正されている。

========
トリガー
========

ユーザーが鑑賞の訂正をシステムに指示する。

==========
基本フロー
==========

#. ユーザーは訂正する鑑賞を指定する。
#. ユーザーは鑑賞の訂正内容を提示する。
#. システムは鑑賞を訂正する。

==========
例外フロー
==========

なし。

==============
ビジネスルール
==============

なし。
